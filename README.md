# Footprint Guard

청소년이 SNS 게시 전 텍스트와 이미지에 포함된 개인정보·디지털 발자국 후보를 확인하고 안전하게 수정하도록 돕는 프로젝트입니다.

## 로컬 실행

```bash
bun install
bun run dev
```

프로덕션 빌드는 `bun run build`, 테스트는 `bun test`, 정적 검사는 `bun run lint`로 실행합니다.

## AI 설정

`.env.example`을 참고해 로컬 `.env`에 서버 전용 `LOVABLE_API_KEY`를 설정하세요. 모델 변경이 필요할 때만 `LOVABLE_MODEL`을 지정합니다. 키를 `VITE_` 접두사 변수에 넣으면 브라우저 번들에 노출될 수 있으므로 사용하지 마세요.

키가 없거나 AI 호출·응답 검증에 실패하면 앱은 데모 결과임을 명시하고 대체 결과를 표시합니다. 이미지 OCR과 가림 처리는 브라우저 안에서 수행하며 원본 이미지를 서버로 보내지 않습니다.

## 이미지 자동 탐지 범위

- Tesseract.js 한국어·영어 OCR 기반 전화번호, 이메일, 우편번호, 차량번호, 주소, 학교·소속, SNS 계정 후보
- 지원 브라우저의 Shape Detection API 기반 얼굴·QR코드 후보
- 자동 탐지가 지원되지 않거나 놓친 영역을 위한 수동 선택

자동 결과는 개인정보 후보를 제안하는 보조 기능입니다. 다운로드와 게시 전에 사용자가 직접 최종 이미지를 확인해야 합니다.
